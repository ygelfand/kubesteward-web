# KubeSteward

Manage your Kubernetes Clusters with ease.

This repository contains the source code for the official KubeSteward landing page.

## Features

- **Centralized Access**: Unified management for all your Kubernetes clusters.
- **Enhanced Security**: Enterprise-grade security protocols and access controls.
- **Terminal & SQL Sessions**: Native interactive sessions for direct cluster interaction.

## Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **UI Library**: [Mantine UI](https://mantine.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
